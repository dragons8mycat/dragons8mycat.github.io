Implementation Notes

What is wired today

  • Static frontend suitable for GitHub Pages.
  • Workbook export script generates the JSON used by this UI.
  • SQL files define the `data_lifecycles_mvp` schema and baseline tables for Supabase/Postgres.
  • When network access to Supabase is available, the schema and data importer can be applied without changing the UI model.
